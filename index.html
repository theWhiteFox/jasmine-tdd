What is TDD?

  • Don't write a line of new code unless you first have a failing automated test.
  • Eliminate duplication.

Why practice TDD?

  • Documents behavior of code
    • Easy for new developers to join project
    • Stays up to date, unlike comments
  • Leads to simple designs
    • Solution uses minimal amount of code
    • Tests are first client of code
  • Allows refactoring with confidence
    • Code improves over time
  • Reduces number of defects
    • Maintainable code

How to practice TDD?

Red Green Refactor

Write a short failing test

  • Watch your test fail

Make the test pass in the simplest way possible

  • Nothing too fancy
  • Duplicate code, if needed

Eliminate duplication introduced in the previous step

  • Improve code without changing behavior
  • Covered by tests
  • Don't forget!

Testing Asynchronous Code

Jasmine passes a done function as an argument to each function passed to it. 


it('takes done as an argument', function (done) {

});

Testing Asynchronous Code

A test will not pass until done has been called. 


it('takes done as an argument', function (done) {
  done();
});

Testing Asynchronous Code

By default, Jasmine will wait five seconds for done to be called before failing a test 


it('will pass', function (done) {
  setTimeout(function () {
    done();
  }, 4000);
});

it('will fail', function (done) {
  setTimeout(function () {
    done();
  }, 6000);
});

Top Contributers

Let's test drive a sample angular application that shows the list of top github users in Dublin.

Testing promises


describe('UsersController', function () {
  var $scope, githubService, githubDeferred;
  beforeEach(module('topContributors'));

  beforeEach(inject(function ($rootScope, $controller, $q,
                              _githubService_) {
    $scope = $rootScope.$new();
    githubService = _githubService_;

    githubDeferred = $q.defer();
    spyOn(githubService, 'fetchUsers')
            .and.returnValue(githubDeferred.promise);

    $controller('UsersController');
  }));
});

Testing promises


describe('UsersController', function () {
  // setup

  describe('when the controller starts', function () {
    it('fetches the users from GitHub', function () {
      githubDeferred.resolve([{login: 'testUser'}]);
      $scope.$apply();

      expect($scope.users).toEqual([{login: 'testUser'}]);
    });
  });
});

Testing ajax calls


describe('githubService', function () {
  var githubService;
  beforeEach(module('topContributors'));

  beforeEach(inject(function ($httpBackend,
                              _githubService_) {
    githubService = _githubService_;
  }));

  // tests
});

Testing ajax calls


describe('githubService', function() {
  // setup

  describe('#getUsers', function() {
    it('gets users from GitHub', function() {
      $httpBackend
        .expectGET('https://api.github.com/search/users' +
        '?q=location:Dublin&sort=followers')
        .respond({items: ['user']});

      githubService.getUsers().then(function(response) {
        expect(response).toEqual(['user']);
      });

      $httpBackend.flush();
    });
  });
});
